//Diagrama de adolfo


Cancelar Pasaje

El cliente informa que desea cancelar un pasaje.
EL agente de ventas da de baja la venta.
El agente de ventas pone el pasaje de nuevo a la venta. 


Realizar reintegro

EL agente de ventas calcula el porcentaje que se le puede llegar a reintegrar al cliente por la cancelacion del viaje.
Se le notifica al cliente el reintegro que se le va a hacer, dependiendo la fecha de cancelacion.


Consultar Viajes

El cliente consulta por distintos los viajes que realiza la empresa y por las promociones disponibles.
El agente de ventas le informa todos los viajes que realiza la empresa,las promociones existentes y las fechas disponibles.
El cliente selecciona un viaje de entre todos los viajes que el agente de ventas le ofrece.


Reservar pasaje

El agente de ventas le informa las habitaciones disponibles del crucero que va a realizar dicho viaje.
El cliente elije la habitacion para realizar el viaje.
El agente reserva el pasaje y la habitacion asociada al cliente


Tomar datos

El agente de ventas le solicita los datos personales. 
EL cliente le da los datos personales al agente de venta.
El agente de ventas guarda los datos del cliente


Registrar Pago

El cliente solicita pagar el pasaje adquirido.
El agente de ventas le informa el total que debe abonar para pagar el pasaje.
El cliente paga el pasaje.
El agente de ventas registra el pago


Realizar viaje

El area de operaciones informa que el viaje ha sido realizado.
El area de operaciones informa los consumos adicionales que ha solicitado el cliente.


Registrar pago de servicios

El agente de ventas informa al cliente los servicios adicionales que el cliente a solicitado
El cliente abona los servicios adicionales.
El agente de ventas registra el pago de los mismos



Llenar encuesta

El cliente llena la encuesta sobre el viaje.
El agente de ventas recibe la encuesta completada por el cliente.
